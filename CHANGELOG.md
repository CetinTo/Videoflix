# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-12

### Added
- Initial release of Videoflix backend
- User authentication with JWT and HTTP-Only cookies
- Email activation and password reset
- Video upload and management
- Automatic HLS conversion (360p, 480p, 720p, 1080p)
- Thumbnail generation
- FFmpeg integration for video processing
- Django-RQ for asynchronous tasks
- PostgreSQL database support
- Redis caching and message queue
- API documentation with Swagger/ReDoc
- Docker and Docker Compose configuration
- Django Debug Toolbar integration
- Custom user model with email authentication
- Video categorization and ratings
- Watch history tracking
- Favorites system
- Admin panel customization

### Security
- HTTP-Only cookies for token storage
- CSRF protection
- Email-based authentication
- Secure password reset flow

### Documentation
- Comprehensive README
- Installation guide
- Quick start guide
- API documentation
- Deployment guide
- Security policy
- Contributing guidelines
